# Business Performance Data Project
This analysis explores the performance of a City Bike rental services using historical services data, aiming to gain insight into how different regions, most popular pick up location and age groups that rent most bike contribute to revenue and identify areas for scaling or improvement. 
Using Microsoft Excel, data was collected, cleaned, aggregated and visualized through Power Query, Pivot Tables, Charts, and Slicers to identify high-performing areas and improvement opportunities. 
Data cleaning and transformation were carried out using Excel’s Power Query. Pivot Tables were used to aggregate and filter the data, while Charts and Slicers supported effective visual exploration and pattern identification across different business dimensions.
